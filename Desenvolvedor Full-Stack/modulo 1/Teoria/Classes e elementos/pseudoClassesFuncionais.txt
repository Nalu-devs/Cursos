Pseudoclasses funcionais permitem selecionar elementos com base em condições ou relacionamentos
mais complexos. Ao contrário das pseudoclasses comuns, que selecionam elementos com base em um 
estado, por exemplo, :hover, :focus, as pseudoclasses funcionais aceitam argumentos entre parênteses, 
daí o nome "pseudoclasses funcionais".

Exemplos de pseudoclasses funcionais são:
:is()
:where()
:has()
:not()

A :is() pseudoclasse é útil quando você deseja estilizar um grupo de elementos que compartilham 
algumas, mas não todas, as características. Por exemplo, você pode querer estilizar diferentes tipos 
de botões no seu site, incluindo buttonelementos, links estilizados como botões e inputelementos com 
os tipos submite reset. Aqui está um exemplo em HTML que representa isso:
<button>Click Me</button>
<a href="#" class="button">Click Me Too</a>
<input type="submit" value="Submit" />
<input type="reset" value="Reset" />

Sem a :is()função, você teria que escrever um seletor complexo como este:
button,
a.button,
input[type='submit'],
input[type='reset'] {
  background-color: darkblue;
  color: white;
  border: 1px solid darkblue;
  padding: 10px 20px;
  text-decoration: none;
  border-radius: 5px;
  cursor: pointer;
  display: inline-block;
  margin: 5px;
  font-size: 16px;
  text-align: center;
}
button:hover,
a.button:hover,
input[type='submit']:hover,
input[type='reset']:hover {
  background-color: lightblue;
  border-color: lightblue;
}

Com a :is()função, você pode escrever um seletor mais compacto e compreensível como este:
:is(button, a.button, input[type='submit'], input[type='reset']) {
  background-color: darkblue;
  color: white;
  border: 1px solid darkblue;
  padding: 10px 20px;
  text-decoration: none;
  border-radius: 5px;
  cursor: pointer;
  display: inline-block;
  margin: 5px;
  font-size: 16px;
  text-align: center;
}
:is(button, a.button, input[type='submit'], input[type='reset']):hover {
  background-color: blue;
  border-color: blue;
}

A :where() pseudoclasse funciona de forma semelhante a :is(), mas não aumenta a especificidade dos 
seus seletores. Isso a torna ideal para aplicar estilos sem afetar a especificidade de outras regras.
Por exemplo, você pode usar a :where()função para aplicar zero margine paddinga elementos de título. 
Isso garante que a redefinição não interfira em estilos mais específicos que você possa aplicar 
posteriormente. Aqui está o HTML para isso:
<h1>Page Title</h1>
<h2>Subtitle</h2>
<h3>A point</h3>
