O CSS fornece diversas pseudoclasses de entrada que podem tornar seus formulários mais intuitivos 
e fáceis de usar. Vamos analisar essas pseudoclasses em detalhes.

A :focus pseudoclasse permite que você direcione um elemento de entrada quando um usuário o 
seleciona ou clica, chamando a atenção para o campo de entrada ativo. Isso ajuda os usuários 
a identificar facilmente onde estão digitando no momento:
input:focus {
  border: 2px solid crimson;
  outline: none;
}

Como o nome indica, a :hover pseudoclasse é acionada quando o usuário posiciona o cursor sobre um 
elemento. Ela fornece um feedback visual antes que o usuário interaja com a entrada, alertando-o 
efetivamente de que a entrada está pronta para ação.
input:hover {
  background-color: orange;
}

A :checked pseudoclasse permite estilizar um botão de opção ou caixa de seleção de forma diferente 
quando um usuário os seleciona. Dessa forma, o usuário pode ver facilmente qual opção escolheu.
<form>
  I have read the terms<input class="checkbox" type="checkbox">
</form>
.checkbox:checked {
  appearance: none;
  width: 12px;
  height: 12px;
  background-color: coral;
}

:required direciona elementos de entrada que possuem o required atributo. Ele sinaliza ao usuário 
que ele deve preencher o campo para enviar o formulário.
O HTML abaixo tem um formulário com dois elementos de entrada obrigatórios:
<form action="#">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required />

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required />

  <label for="phone">Phone Number:</label>
  <input type="tel" id="phone" name="phone" />

  <button type="submit">Submit</button>
</form>
input:required {
  border: 2px solid orange;
}

Ao validar formulários, você pode usar a :valid pseudoclasse para estilizar os campos de entrada 
que atendem aos critérios de validação e :invalidos campos que não atendem aos critérios. 
Normalmente, você usará verde para uma entrada válida e vermelho para uma entrada inválida.
Aqui está um exemplo usando a :valid pseudoclasse:
input:valid {
  border-color: green;
}

Aqui está um exemplo de uso da :invalid pseudoclasse:
input:invalid {
  border-color: crimson;
}

A :disabled pseudoclasse permite selecionar e estilizar elementos de entrada que estão desabilitados. 
Esses elementos possuem o disabledatributo , que impede os usuários de interagir com eles. 
Quando uma entrada está desabilitada, ela não pode ser clicada, focalizada ou editada.
<label for="name">Name:</label>
<input class="text-input" type="text" id="name" name="name" disabled />
.text-input:disabled {
  background-color: lightgray;
  cursor: not-allowed;
}

O cursor: not-allowed;valor da propriedade CSS altera a aparência do cursor para indicar que uma 
ação não é permitida.
Quando aplicado a um elemento, ele mostra um cursor com um símbolo de círculo e barra (geralmente 
um círculo com uma linha diagonal atravessando-o) para sinalizar aos usuários que o elemento não é 
interativo ou não pode ser clicado ou interagido.

Aqui estão alguns outros exemplos de pseudoclasses de entrada e o que elas fazem:

:autofill: aplica estilos aos campos de entrada que o navegador preenche automaticamente com dados 
salvos.
:optional: estilos de entrada de elementos que não são necessários e podem ser deixados em branco.
:in-rangee :out-of-range: elementos de estilo com base em se seus valores estão dentro ou fora das 
restrições de intervalo especificadas.
