Pseudoclasses de estrutura em árvore permitem que você direcione e estilize elementos com base 
em sua posição na árvore do documento. A árvore do documento refere-se à estrutura hierárquica dos 
elementos em um documento HTML.

Aqui está uma lista de pseudoclasses de estrutura de árvore:
:root
:empty
:nth-child(n)
:nth-last-child(n)
:first-child
:last-child
:only-child
:nth-of-type
:first-of-type
:last-of-type
:only-of-type

A :root pseudoclasse geralmente é o htmlelemento raiz. Ela ajuda você a atingir o nível mais alto
do documento, permitindo aplicar um estilo comum a todo o documento.
:root {
  background: black;
  color: aliceblue;
}
A :root pseudoclasse também é comumente usada na configuração de variáveis ​​CSS:
:root {
  --main-font: 'Arial, sans-serif';
  --primary-color: blue; 
  --secondary-color: green; 
}

Elementos vazios, ou seja, elementos sem filhos além de espaços em branco, também são incluídos 
na árvore de documentos. É por isso que existe uma :emptypseudoclasse para lidar com elementos vazios. 
Por exemplo, este código HTML tem dois itens de lista vazios:
<ul>
  <li>Item 1</li>
  <li></li> <!-- This list is empty -->
  <li>Item 2</li>
  <li></li> <!-- Another empty list -->
  <li>Item 3</li>
</ul>
:empty {
  background: black;
}

:nth-child(n) permite selecionar elementos com base em sua posição dentro de um elemento pai, 
enquanto :nth-last-child(n) permite selecionar elementos contando a partir do final. 
O npode ser um número específico ou uma palavra-chave como oddou even. Isso é extremamente útil 
para estilizar células de tabela com base na posição: pares e ímpares.
<table>
  <tr>
    <th>Item</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Apple</td>
    <td>$1.00</td>
  </tr>
  <tr>
    <td>Banana</td>
    <td>$0.50</td>
  </tr>
  <tr>
    <td>Orange</td>
    <td>$0.80</td>
  </tr>
</table>
th,
td {
  border: 1px solid lightgray;
  padding: 8px;
}

tr:nth-child(even) {
  background-color: orangered;
}

tr:nth-child(odd) {
  background-color: lightgreen;
}

As pseudoclasses, reagem em itens dentro de um contêiner pai ou em todo o :first-child 
documento :last-child.:only-child
:first-child seleciona o primeiro elemento em um elemento pai ou no documento.
:last-child seleciona o último elemento em um elemento pai ou no documento.
:only-child seleciona o único elemento em um elemento pai ou no documento.
Usar as :first-child pseudoclasses :last-childe selecionará ambos Item 1 e Item 3 neste HTML:
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
li:first-child {
  background-color: orangered;
}

li:last-child {
  background-color: lightgreen;
}
ul li:first-child {
  background-color: orangered;
}

ul li:last-child {
  background-color: lightgreen;
}

Para mostrar como a :only-child pseudoclasse funciona, aqui está um exemplo HTML com dois 
div elementos separados:
<div class="container">
  <div>This is the only item in this container.</div>
</div>

<div class="container">
  <div>This is one of two items in this container.</div>
  <div>Here is the second item.</div>
</div>
.container div:only-child {
  border: 2px solid crimson;
  padding: 10px;
  background-color: lightblue;
}

As :first-of-type pseudoclasses :last-of-type e selecionam a primeira e a última ocorrência de um 
tipo de elemento específico dentro de seu elemento pai. Elas são úteis para aplicar estilos 
exclusivos à primeira ou à última instância desse tipo de elemento entre seus irmãos.
No HTML abaixo, :first-of-type e :last-of-type se aplica ao primeiro elemento e ao último 
elemento dentro do sectionelemento:
<section>
  <h2>Introduction</h2>
  <p>This is the first paragraph in the first section</p>
  <p>This is the second paragraph in the first section</p>
</section>
<section>
  <h2>Details</h2>
  <p>This is the first paragraph in the second section.</p>
  <p>This is the second paragraph in the second section.</p>
</section>
section p:first-of-type {
  background-color: lightgreen;
}

section p:last-of-type {
  background-color:lightblue;
}

:nth-of-type(n) permite selecionar um elemento específico dentro de seu elemento pai com base em 
sua posição entre irmãos do mesmo tipo. Por exemplo, no HTML abaixo, :nth-of-type(2) direciona 
para o segundo elemento no contêiner:
<div class="container">
  <p>First paragraph</p>
  <h2>First heading</h2>
  <p>Second paragraph</p>
  <p>Third paragraph</p>
  <h2>Second heading</h2>
</div>
p:nth-of-type(2) {
  color: red;
  font-weight: bold;
}

:only-of-typeseleciona um elemento se ele for o único de seu tipo dentro do elemento pai. 
Isso pode ser útil para enfatizar itens individuais ou garantir que eles tenham um estilo diferente
quando não fizerem parte de um grupo.
No HTML abaixo, há dois divelementos, sendo que um deles possui um único elemento:
<div class="container">
  <p>The only paragraph</p>
</div>

<div class="container">
  <p>The first paragraph</p>
  <p>The second paragraph</p>
</div>
p:only-of-type {
  border: 4px solid green;
}
