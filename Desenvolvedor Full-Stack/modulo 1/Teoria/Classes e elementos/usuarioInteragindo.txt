A :active pseudoclasse aplica estilos quando um elemento é ativado pelo usuário. 
Por exemplo, quando o usuário clica em um botão ou link, ela fornece um feedback visual imediato,
mostrando aos usuários que suas ações estão sendo reconhecidas.
a:active {
  color: crimson;
}

A :hover pseudoclasse é acionada quando um usuário passa o mouse ou outro dispositivo apontador 
sobre um elemento. Desenvolvedores costumam usá-la para criar feedback visual para botões, links 
ou qualquer elemento que deva responder à atenção do usuário. Aqui está um botão sobre o qual o 
usuário passaria o mouse antes de clicar:
<button class="btn">Hover Over Me</button>
.btn:hover {
  background-color: darkgreen;
  color: white;
  cursor: pointer;
}

A :focus pseudoclasse aplica estilos quando um elemento ganha foco, normalmente por meio da navegação
pelo teclado ou quando um usuário clica em uma entrada de formulário. Isso não serve apenas para 
feedback, mas também é crucial para a acessibilidade. Ele garante que usuários que dependem muito 
do teclado possam identificar facilmente com qual elemento estão interagindo.
<form>
  <input type="text" />
</form>
input:focus {
  outline: 2px solid darkgreen;
  border-radius: 4px;
}

A :visited pseudoclasse tem como alvo um link que o usuário visitou. Isso pode ser útil para ajudar 
os usuários a distinguir entre páginas que já visitaram e aquelas que ainda não visitaram. Aqui 
está um exemplo de alteração da cor do texto âncora para ciano quando o link é visitado:
a:visited {
  color: cyan;
}

A :checked pseudoclasse em CSS permite estilizar elementos de formulário, como caixas de seleção 
e botões de opção, quando selecionados (marcados). Essa pseudoclasse é útil para personalizar a
aparência desses elementos e aprimorar a experiência do usuário, mesmo que os navegadores forneçam 
estilos padrão para eles.
<form>
  Agree <input class="checkbox" type="checkbox" />
</form>
.checkbox:checked {
  appearance: none;
  width: 12px;
  height: 12px;
  background-color: red;
}

Outros exemplos de pseudoclasses de ação são:
:focus-within: para aplicar estilos a um elemento quando ele ou qualquer um de seus descendentes 
tiver foco.
:enabled: para direcionar botões de formulário ou outros elementos que estão atualmente habilitados.
:disabled: para direcionar botões de formulário ou outros elementos que estão desabilitados.
:target: para aplicar estilos a um elemento que é o alvo de um fragmento de URL 
(a parte de uma URL após o #símbolo).
