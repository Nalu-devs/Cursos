Teoria das Cores
Definição de Teoria das Cores : É o estudo de como as cores interagem entre si e como afetam nossa 
percepção. Abrange as relações entre as cores, a harmonia das cores e o impacto psicológico das 
cores.

Cores primárias : essas cores, amarelo, azul e vermelho, são os matizes fundamentais dos quais 
todas as outras cores são derivadas.

Cores secundárias : essas cores resultam da mistura de quantidades iguais de duas cores primárias. 
Verde, laranja e roxo são exemplos de cores secundárias.

Cores terciárias : essas cores resultam da combinação de uma cor primária com uma cor secundária 
adjacente. Amarelo-esverdeado, azul-esverdeado e azul-violeta são exemplos de cores terciárias.

Cores quentes : essas cores, que incluem vermelho, laranja e amarelo, evocam sentimentos de 
conforto, calor e aconchego.

Cores frias : essas cores, que incluem azul, verde e roxo, evocam sentimentos de calma, serenidade 
e profissionalismo.

Círculo de Cores : O círculo cromático é um diagrama circular que mostra como as cores se relacionam.
É uma ferramenta essencial para designers, pois os ajuda a selecionar combinações de cores.

Esquemas de cores análogas : esses esquemas de cores criam experiências coesas e relaxantes. Eles 
têm cores análogas, adjacentes umas às outras no círculo cromático.

Esquemas de Cores Complementares : Esses esquemas de cores criam alto contraste e impacto visual.
 Suas cores estão localizadas em extremidades opostas do círculo cromático, uma em relação à outra.

Esquema de Cores Triádico : Este esquema de cores possui cores vibrantes. É composto por cores 
aproximadamente equidistantes entre si. Se conectadas, formam um triângulo equilátero no círculo 
cromático.

Esquema de Cores Monocromático : Neste esquema de cores, todas as cores são derivadas da mesma 
cor base, ajustando sua claridade, escuridão e saturação. Isso evoca uma sensação de unidade e 
harmonia, ao mesmo tempo em que cria contraste.

Diferentes maneiras de trabalhar com cores em CSS

Cores Nomeadas : Essas cores são nomes de cores predefinidos reconhecidos pelos navegadores. Exemplos incluem blue, darkred, lightgreen.
rgb()Função : RGB significa Vermelho, Verde e Azul — as cores primárias da luz. Essas três cores 
são combinadas em diferentes intensidades para criar uma ampla gama de cores. A rgb()função permite
definir cores usando o modelo de cores RGB.
Exemplo de código
p {
  color: rgb(255, 0, 0);
}

rgba()Função : Esta função adiciona um quarto valor, alfa, que controla a transparência da cor. 
Se não for fornecido, o valor alfa padrão será 1.
Exemplo de código
div {
  background-color: rgba(0, 0, 255, 0.5);
}

hsl()Função : HSL significa Matiz, Saturação e Luminosidade — três componentes principais que 
definem uma cor.
Exemplo de código
p {
  color: hsl(120, 100%, 50%);
}

hsla()Função : Esta função adiciona um quarto valor, alfa, que controla a opacidade da cor.
Exemplo de código
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}

Hexadecimal : Um código hexadecimal (abreviação de código hexadecimal) é uma sequência de seis 
caracteres usada para representar cores no modelo de cores RGB. O "hex" refere-se ao sistema de 
numeração de base 16, que usa dígitos de 0 a 9 e letras de A a F.
Exemplo de código
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}

Gradientes lineares e radiais
Gradientes Lineares : Esses gradientes criam uma mistura gradual entre cores ao longo de uma linha 
reta. Você pode controlar a direção dessa linha usando palavras-chave como to top, to right, 
to bottom right, ou ângulos como 45deg. Você pode usar qualquer cor CSS válida e quantos intervalos 
de cor desejar.
Exemplo de código
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}

Gradientes radiais : esses gradientes criam gradientes circulares ou elípticos que irradiam de um 
ponto central.
Exemplo de código
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
