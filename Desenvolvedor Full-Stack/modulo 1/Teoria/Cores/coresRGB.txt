Ao trabalhar com cores em CSS, é essencial entender o modelo de cores RGB. RGB significa 
Vermelho, Verde e Azul — as cores primárias da luz. Essas três cores são combinadas em diferentes 
intensidades para criar uma ampla gama de cores.
O modelo de cores RGB é um modelo de cores aditivo, o que significa que as cores são criadas pela 
combinação de luz em intensidades variadas.

A intensidade de cada cor pode variar de 0 (o que significa ausência de luz) a 255 (o que significa 
luz total). Ao misturar diferentes níveis de vermelho, verde e azul, você pode produzir qualquer 
cor que veja na tela.
Em CSS, a rgb()função permite definir cores usando o modelo de cores RGB. A função assume três 
valores, cada um representando a intensidade da luz vermelha, verde e azul, respectivamente. 
Aqui está a sintaxe básica:
element {
  color: rgb(red, green, blue);
}

O CSS também fornece a rgba()função, que adiciona um quarto valor —alpha— que controla a 
transparência da cor. O valor alfa varia de 0(completamente transparente) a 1(completamente opaco). 
Aqui está um exemplo de uso rgba():
div {
  background-color: rgba(0, 0, 255, 0.5);
}

O modelo de cores RGB é especialmente útil para mídia digital porque corresponde diretamente à forma 
como as telas exibem as cores. Monitores e displays usam pequenos pixels vermelhos, verdes e azuis 
para criar as cores que você vê. Ao controlar a intensidade desses pixels por meio de valores RGB, 
você pode obter uma ampla gama de cores para o seu web design.
