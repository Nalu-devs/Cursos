Gradientes em CSS permitem criar transições suaves entre duas ou mais cores especificadas.
Eles são frequentemente usados ​​para adicionar profundidade, textura e interesse visual a designs
web sem a necessidade de arquivos de imagem.

Gradientes lineares criam uma mistura gradual entre cores ao longo de uma linha reta. Você pode 
controlar a direção dessa linha e as cores utilizadas. Aqui está um exemplo básico de um gradiente 
linear:
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}

Você também pode usar várias paradas de cores em um gradiente linear:
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}

Gradientes radiais, por outro lado, criam gradientes circulares ou elípticos que irradiam de um 
ponto central. Veja um exemplo:
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
