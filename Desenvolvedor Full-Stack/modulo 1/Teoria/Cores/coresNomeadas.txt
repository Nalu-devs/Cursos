Uma das maneiras mais simples de definir cores em CSS é usando cores nomeadas. Cores nomeadas 
são nomes de cores predefinidos reconhecidos pelos navegadores. Aqui está um exemplo do uso de 
uma cor nomeada para um elemento de parágrafo:
p {
  color: red;
}

Cores nomeadas em CSS são uma coleção de 148 nomes de cores padrão Esses nomes são simples de 
usar e tornam seu código mais legível, além de serem autodescritivos.

Cores nomeadas em CSS são práticas, mas limitadas, com apenas 148 opções disponíveis. 
Elas podem não oferecer o tom preciso necessário para designs mais detalhados.
