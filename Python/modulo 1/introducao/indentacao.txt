INDENTAÇÃO

No Python, a indentação (espaços ou tabulações no início de uma linha) é utilizada para delimitar 
blocos de código. Diferente de outras linguagens que utilizam chaves ou palavras-chave, o Python 
utiliza a indentação para determinar o escopo das declarações.
EXEMPLO:

if condition:
    # Bloco de código se a condição for verdadeira
    instrucao1
    instrucao2

else:
    # Bloco de código se a condição for falsa
    instrucao3
    instrucao4

####É fundamental manter uma indentação consistente em todo o código para evitar erros de sintaxe####
